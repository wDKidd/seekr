# README

## Setup

## 1. Setup Ruby on Rails

##### Use this guide for mac or linux:
##### https://gorails.com/setup/ubuntu/15.04


## 2. In the terminal run
```
$ git clone https://github.com/wDKidd/seekr.git
```
#### then

```
$ cd seekr

$ bundle install
```
#### start the server
```
$ rails server
```
